# AI автоматизації — MAD team

Дашборд для аналізу портфеля AI-автоматизацій технічної команди **MAD team**.
Показує статуси, фінансовий ефект, роадмап та зведену таблицю автоматизацій.

## 🔗 Демо

Дашборд відкривається за посиланням GitHub Pages (додається після публікації).

> **Важливо:** сайт **не містить жодних даних**. Це порожній дашборд.
> Дані ви завантажуєте у свій браузер локально (кнопка «📂 Обрати файл»),
> вони нікуди не передаються і залишаються тільки у вас.

## Як користуватися

1. Відкрийте дашборд (посилання вище або файл `index.html` локально).
2. Натисніть **«📂 Обрати файл»** і виберіть свій Excel (`.xlsx`).
3. Дашборд автоматично порахує показники й побудує графіки.
4. Кнопка **«Зберегти звіт»** вивантажує статичний HTML-знімок із поточними фільтрами.

## Структура Excel-файлу

Файл має містити три аркуші.

### Аркуш «Автоматизації»

| Колонка | Опис |
|---|---|
| Номер задачі (Jira) | ID задачі (напр. `MAD-101`) — посилання формується автоматично |
| Назва автоматизації | Коротка назва |
| Підрозділ | Замовник автоматизації |
| Інструмент | Технологія/платформа (n8n, Claude API тощо) |
| Статус | `Виконано`, `В розробці`, `Тестування`, `Discovery`, `В черзі`, `Hold` |
| FTE | Ефект в одиницях FTE (напр. `0.25` = чверть ставки) |
| Розробник | Хто реалізував |
| Фото шлях | Шлях/URL до фото розробника (для роадмапу) |
| Дата створення | Коли задачу поставлено |
| Дата реалізації | Коли автоматизацію запущено |
| Опис | Короткий опис |
| Спільнота? | `так` / `ні` — ініціатива спільноти |
| AI-агентів, шт. | Кількість AI-агентів |
| Процеси з AI-агентами, шт. | Кількість процесів |

### Аркуш «Довідник»

| Параметр | Значення |
|---|---|
| Вартість 1 FTE (грос/міс), грн | Місячна ставка для розрахунку ефекту (напр. `33000`) |
| Jira base URL | Базова адреса для посилань (напр. `https://company.atlassian.net/browse/`) |

### Аркуш «Інструкція»

Текстова підказка для людини, що заповнює файл (дашборд її не читає).

## Розрахунок ефекту

```
Річний ефект = FTE × Вартість 1 FTE (грос/міс) × 12
```

Фінансовий ефект рахується **тільки** для задач зі статусом `Виконано`.

## Технології

Чистий HTML + JavaScript, без бекенду. Усі обчислення — у браузері.

- [Chart.js](https://www.chartjs.org/) — графіки
- [SheetJS (xlsx)](https://sheetjs.com/) — читання Excel
